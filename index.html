configglue?

configglue is a library that glues together Python's OptionParser and ConfigParser, so that you don't have to repeat yourself when you want to export the same options to a configuration file and a command-line interface.

configglue - main features

  • ini-style configuration files
  • out of the box hierarchical configuration support
  • schema-based configuration
  • command-line integration
  • configuration validation
  • basic types included
  • easy to extend with custom types
  • bonus: django integration via django-configglue
